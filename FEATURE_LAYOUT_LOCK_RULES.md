# Credit Vivo Layout Lock Rules

Any new parser/scanner feature must not change the approved public pages.

## Allowed
- Add backend files
- Add API client files
- Add dashboard-only scanner upload behavior
- Add admin-only review tables
- Add hidden/internal routes
- Add environment variables
- Add docs/prompts

## Not allowed
- Redesign homepage
- Change public nav
- Change hero layout
- Change public typography
- Add scanner jargon to public homepage
- Add public claims about guaranteed score increases or removals
- Add Credit Vivo Native Parser branding to customer-facing pages

This version removes Anthropic/Claude and PyMuPDF dependencies. It uses Credit Vivo Native Parser with no paid AI API.
