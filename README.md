# Credit Vivo Emergent Preview

This branch contains the Emergent `credit-boost-ai-2` code preserved for Credit Vivo integration review.

Live visual preview:
https://frontend-three-livid-22.vercel.app/

## Included

- `frontend/` - React / Craco customer-facing app
- `backend/` - FastAPI prototype backend from Emergent
- `memory/PRD.md` - product notes
- `EMERGENT_INTEGRATION_AUDIT.md` - Codex audit notes
- `INTEGRATION_README.md` - integration guidance

## Not Included

- `.env` files
- API keys or passwords
- `node_modules`
- local build output
- customer data

## Status

This is for review and selective integration only. It is not production-ready.

Recommended path: port the useful Emergent pages and workflows into `Credit-Vivo-V2` carefully, instead of replacing the current app all at once.
